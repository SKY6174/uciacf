### 2. 애로사항 및 해결과정 (Troubleshooting)
* **로컬 Docker 미구동 이슈**:
  * **애로사항**: `npx supabase status`를 이용한 로컬 컨테이너 상태 점검 시, Docker 데몬이 실행 중이지 않아 CLI 명령어가 에러를 발생시켰습니다.
  * **해결과정**: 로컬 기동 대신 `supabase/.temp/linked-project.json` 파일을 정밀 검사하여 연결된 원격 프로젝트 Ref(`tbbdcjqgzoqwddgvywph`) 및 원격 URL 정보를 정확히 추출하여 연동을 안전하게 정상화하였습니다.
* **조직도 선 어긋남 현상**:
  * **애로사항**: 고정 px 및 %로 연결선을 렌더링 시 브라우저 줌이나 폰트 크기 변경에 따라 격자 선이 어긋나거나 끊겨서 표시되었습니다.
  * **해결과정**: Flexbox의 자식 항목 자체 가상 요소를 50% 분할하는 표준 정렬 기법을 적용하여 카드가 늘어나더라도 선이 유연하게 접착되도록 스타일을 완전히 개정했습니다.
* **Next.js SSR Hydration Mismatch 경고**:
  * **애로사항**: 컴포넌트 초기 렌더링 시 `localStorage`를 직접 읽으려 하면 서버와 브라우저의 마크업 불일치 경고가 콘솔에 도출되었습니다.
  * **해결과정**: 초기 상태는 홈(통합 대시보드)으로 둔 뒤, 렌더링이 완료된 마운트(`useEffect`) 시점에 클라이언트 측에서만 탭을 안전하게 복원하도록 구조화하여 해결했습니다.
* **다중 Level-2 지부로 인한 가로 스크롤 밀림 및 잘림 현상**:
  * **애로사항**: 국책사업단이 Level 2로 승격하여 3개 대형 지부가 가로로 정렬되자 기존 1550px 너비 캔버스에서 가로 폭이 비좁아져 카드가 겹치거나 화면이 어색해졌습니다.
  * **해결과정**: `.org-chart-inner` 캔버스의 최소 너비를 `1800px`로 상향하고, 점선 커넥터 속성을 오버라이드하여 3개 지부 전체가 와이드하고 균형 있게 레이아웃되도록 보완했습니다.
* **부단장의 구석 정렬로 인한 전체 대시보드 대칭 훼손**:
  * **애로사항**: Flexbox로 3개 L2 카드를 가로 정렬하자 단장이 중앙에 서고 부단장이 좌측으로 쏠려 수직 기둥 축이 틀어지는 부자연스러운 현상이 발견되었습니다.
  * **해결과정**: 단장-부단장을 메인 기둥(1020px 고정)으로 묶고, 우측 지부들을 절대 좌표 분기(`position: absolute`)와 전용 기하학적 교량선(`.org-l2-horizontal-bridge`)으로 라우팅하여 0.1픽셀 오차도 없는 수직/수평 정렬 기하학을 구축해 해결했습니다.
